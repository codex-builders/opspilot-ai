# OpsPilot AI

Read-only Sev-1 incident assistant that combines mock enterprise evidence and prepares drafts for human review.

## Practical Deliverables

- `docs/researched_requirements.md` - EU regulatory research.
- `docs/recommended_updates.md` - recommended architecture, data flow, business requirements, NFRs, and priorities.
- `docs/architecture.md` and `docs/data-flow.md` - implementation diagrams and workflow documentation.
- `docs/test-plan.md` and `docs/demo-script.md` - final testing and presentation material.
- `outputs/incident-summary.md`, `outputs/teams-update.md`, `outputs/cab-brief.md`, and `outputs/post-incident-report.md` - expected demo outputs.

All generated outputs must include evidence references and remain drafts until an authorised human approves them.
